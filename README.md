## Project Description
This project is developed to demonstrate the concepts of Unit Testing and Integration Testing using the pytest framework.  
A simple banking application is implemented and tested to verify that individual functions work correctly in isolation and that multiple functions interact properly when combined.

## How to Run Tests
pytest =run all tests
pytest -v=Run Tests with Detailed Output
pytest test_unit.py=Run Specific Test File
generate html report=pytest --html=report.html -v
